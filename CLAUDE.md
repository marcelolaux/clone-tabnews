# CLAUDE.md

## Project Overview

This is a clone of [TabNews](https://tabnews.com.br), a Brazilian tech news and community platform. This project is being built as part of the learning course at https://curso.dev.

## Tech Stack

- **Framework**: Next.js 13
- **Frontend**: React 18
- **Code Formatting**: Prettier

## Project Structure

```
clone-tabnews/
├── pages/           # Next.js pages (file-based routing)
│   └── index.js     # Home page
├── package.json     # Dependencies and scripts
├── .editorconfig    # Editor configuration (2 spaces indentation)
└── .nvmrc           # Node version specification
```

## Development Commands

```bash
# Start development server
npm run dev

# Check code formatting
npm run lint:check

# Fix code formatting
npm run lint:fix
```

## Code Style

- Use 2 spaces for indentation
- Format code with Prettier before committing
- Follow React/Next.js conventions
